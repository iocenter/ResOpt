bool Model::resolvePipeRouting ( )

Checks the current routing of wells and pipes, and connects them correctly.

Wells and Pipes may have an OUTLETPIPE defined. This is either a fixed number, or a routing variable. This function tries to resolve the current value of the OUTLETPIPE to a NUMBER assigned to a Pipe.

Returns:
true if successful, false if not resolved
