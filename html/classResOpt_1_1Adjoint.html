bool ResOpt::Adjoint::perturbStream ( double  eps_x)

Perturbes the Stream asociated with these adjoints.

All the Stream quantities (oil, gas, water, pressure) will be set to: current_value + adjoint * eps_x

Parameters:
eps_xThe size of the perturbation
Returns:
True if everything went ok, false if the asociated Stream could not be found.
