Detailed Description

Model where input rates to all parts of the system are treated as variables.

In this type of Model there is no automatic link between the output of a upstream part and the input to a downstream part. The input rates to every pipe segment in the Model is included as a variable. Constraints are included to honour the mass balance in the system (c = q_in - q_out = 0).
