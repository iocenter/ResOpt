Detailed Description

Launches the project.

This class takes care of launching the project for a set of variable values. The launcher holds a copy of the Model and the ReservorSimulator. Each Launcher lives in a separate thread. Multiple launchers may be run in parallel to exploit the available cores on the computer.

void ResOpt::Launcher::evaluate ( Case *  c) [slot]

Evaluates the project with the variable values defined in the Case c.

First the reservoir simulator is launched, then the pipe network is evaluated. After the project is evaluated, the constraints and objective are updated, and the updated values are copied to the Case c.
