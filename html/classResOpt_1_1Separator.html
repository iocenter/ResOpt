Detailed Description

A separator that can remove phases.

The separator can be fed by any number of pipes, and feeds one upstream pipe. The total rates going through the separator is calculated from the feed pipes. An installation time is set for the separator. From the installation time and on, any of the phases flowing through the separator can be removed.
