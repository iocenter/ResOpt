bool Runner::evaluate ( )

Runs the reservoir simulator on the current model.

This function is called by the optimizer at each iteration to evaluate objective and constraint values. First the simulator input files are generated. Then the simulator is launched, and output from the run is read. Then the pipe network pressures are calculated, and constraint values on separators and well BHPs are calculated. The Model is updated with the results from the run. These results can be read by the optimizer.

bool
